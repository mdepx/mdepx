FMAC
####

**FMAC** (Firmware Media Access Control) module is the primary public interface to the nRF70 Series Wi-Fi driver.
It abstracts the access to the **UMAC** (Upper Media Access Control) firmware in the nRF70 Series Wi-Fi chipset.
